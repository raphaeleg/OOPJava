# What this code does
Play Tic Tac Toe using two clients! Test on a local device.

# Setup
1. Compile Server and Client
`javac Server.java`
`javac Client.java`

2. Run Server
`java Server`

3. Run Client with 2 new command prompt windows.
`java Client`
`java Client`

The player that first submits their name will be set as player 1.
